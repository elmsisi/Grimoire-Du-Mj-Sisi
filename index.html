    Méta

    Comment éditer ce fichier

    Éditer le contenu

    Tout le texte est dans ce fichier index.html, en clair. Ouvre-le avec n'importe quel éditeur de texte (Bloc-notes, VS Code, etc.), cherche le texte à changer, remplace-le, sauvegarde, actualise la page dans ton navigateur.

    Ajouter un chapitre

    1) Ajoute une ligne dans le menu <nav class="toc"> avec un data-target unique.
    2) Ajoute une <section class="chapter" id="chap-TONID"> correspondante plus bas.
    Le data-target du menu doit correspondre à l'id de la section sans le préfixe chap-.

    Héberger sur GitHub Pages

    1. Crée un repo GitHub (public ou privé, peu importe pour toi).
    2. Mets ce fichier dedans, renommé index.html, à la racine.
    3. Va dans Settings → Pages, choisis la branche main et le dossier /root.
    4. GitHub te donne une URL du type https://tonpseudo.github.io/nom-du-repo/ à partager avec tes joueurs (ou juste toi si tu veux le garder privé, en n'activant pas Pages et en le lisant en local).
